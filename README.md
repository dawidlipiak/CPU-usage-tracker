## CPU-usage-tracker

Multithreaded C program for tracking CPU usage on each core based on /proc/stat.

**How to compile program:**
```sh
make
```

**How to run program:**
```sh
./main.out
```

**How to clean build files:**
```sh
make clean
```

**How to build tests program:**
```sh
make test
```

**How to run tests:**
```sh
./tests/test.out
```
